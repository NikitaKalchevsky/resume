Smart DCA Trade Bot

Automated crypto DCA strategy bot for Binance.US, managed entirely via Telegram inline menus — no web interface needed. Scans top coins by volume and momentum, buys on dips below MA20, scales position size with dip depth, and auto-sells on take-profit. Deployed on DigitalOcean with non-blocking async exchange calls.

stack:Python · ccxt (Binance.US) · python-telegram-bot · APScheduler · SQLite · asyncio · DigitalOcean

  • Auto-sell only mode: takes profit on open positions without enabling auto-buy
  • Inline Telegram UI: positions, P&L, settings — full control without a web dashboard
  • DRY_RUN mode for safe strategy testing without real orders
